
LeView source code is distributed under GPL (see COPYING file).

Java files are available in src directory.

You need to download the FreeHEP Java Library (http://java.freehep.org) 
and include it in org/freehep directory.

You need a mmcif directory containing mmcif files that can be downloaded 
from PDBe site http://pdbe.org.
